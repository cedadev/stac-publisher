STAC Publisher
**************

Local deployment
-----------------

1. Install the requirements

    .. code-block::

        pip install -r requirements.txt

2. Copy and update `.stac_publisher.yml.tmp`

    .. code-block::

        cp .stac_publisher.yml.tmp .stac_publisher.yml

3. Run the `stac_publisher`_

    .. code-block::

        python stac_publisher
